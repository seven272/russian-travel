Проектная работа "Путешествие по России"

Стили находятся в соответсвующих папках с названиями блоков,элементов и модификаторов. 
Изображения хранятся в папке images где рассортированы по папкам соотвтесвующим названиям блоков, в которых они используются.
Стили "медиа" подключены в блоках для которых они прменяются. Локально подключен основной шрифт Inter, в качестве альтернативы указаны общедоступные шрифты Arial, sans-serif.
